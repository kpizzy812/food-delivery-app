# Food Delivery App

Мобильное приложение для доставки еды с современным UI и плавными анимациями.

## Особенности

- Поиск ресторанов и блюд с анимированной строкой поиска
- Фильтрация по категориям кухни
- Детальная страница ресторана с меню
- Корзина с управлением товарами
- История заказов
- Оформление заказа с выбором адреса и способа оплаты
- Тактильная обратная связь (Haptics)
- Адаптивный дизайн
- Плавные анимации с Reanimated

## Технологии

- **React Native** - фреймворк для разработки мобильных приложений
- **Expo** - платформа для разработки и сборки React Native приложений
- **Expo Router** - файловая навигация
- **React Native Reanimated** - библиотека для продвинутых анимаций
- **TypeScript** - типизированный JavaScript
- **Expo Haptics** - тактильная обратная связь
- **Context API** - управление состоянием корзины

## Структура проекта

```
food-delivery-app/
├── app/                    # Экраны приложения (Expo Router)
│   ├── (tabs)/            # Вкладки навигации
│   │   ├── index.tsx      # Главный экран
│   │   ├── cart.tsx       # Корзина
│   │   └── orders.tsx     # История заказов
│   ├── restaurant/        # Страница ресторана
│   ├── checkout/          # Оформление заказа
│   └── profile/           # Профиль пользователя
├── components/            # Переиспользуемые компоненты
│   └── ui/               # UI компоненты
├── constants/            # Константы (цвета, размеры)
├── context/              # React Context (корзина)
├── data/                 # Моковые данные

```

## Установка и запуск

### Требования

- Node.js 18+
- npm или yarn
- Expo Go на мобильном устройстве (для тестирования)

### Установка зависимостей

```bash
cd food-delivery-app
npm install
```

### Запуск приложения

```bash
# Запуск в режиме разработки
npm start

# Запуск на Android
npm run android

# Запуск на iOS
npm run ios

# Запуск в веб-браузере
npm run web
```

### Тестирование на реальном устройстве

1. Установите Expo Go на ваше устройство:
   - [Android - Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - [iOS - App Store](https://apps.apple.com/app/expo-go/id982107779)

2. Запустите `npm start`

3. Отсканируйте QR-код в Expo Go

## Сборка APK

### Через Expo (рекомендуется)

```bash
# Установите EAS CLI
npm install -g eas-cli

# Войдите в аккаунт Expo
eas login

# Настройте проект
eas build:configure

# Создайте APK для Android
eas build --platform android --profile preview
```

### Локальная сборка

```bash
# Требуется Android Studio и настроенная среда разработки
expo prebuild
cd android
./gradlew assembleRelease
```

APK будет находиться в: `android/app/build/outputs/apk/release/`

## Основные экраны

### Главный экран
- Анимированная строка поиска
- Фильтрация по категориям
- Список ресторанов с карточками
- Промо-баннеры
- Раздел "Вы заказывали"

### Страница ресторана
- Информация о ресторане
- Меню с категориями
- Карточки блюд
- Модальное окно деталей блюда
- Плавающая кнопка корзины

### Корзина
- Список товаров
- Изменение количества
- Удаление товаров
- Подсчет итоговой суммы
- Переход к оформлению

### Оформление заказа
- Выбор адреса доставки
- Время доставки
- Способ оплаты
- Использование бонусов
- Комментарий к заказу

## UI Компоненты

- **Button** - кнопки с разными вариантами
- **Input** - поля ввода
- **CategoryChip** - чипсы категорий
- **DishCard** - карточки блюд
- **RestaurantCard** - карточки ресторанов
- **Toast** - уведомления
- **PromoBanner** - промо-баннеры
- **ScrollToTopButton** - кнопка прокрутки вверх

## Анимации

- Анимированная строка поиска с перемещением иконки
- Плавные переходы между экранами
- Анимация добавления в корзину
- Модальные окна с жестами
- Тактильная обратная связь при взаимодействии

## Дизайн система

- **Сетка**: 8px система отступов
- **Цвета**: Консистентная цветовая палитра
- **Типографика**: Унифицированные размеры и веса шрифтов
- **Компоненты**: Переиспользуемые UI компоненты
